# Path: 3 Types

## 1. String

- `/about`
- `/`

## 2. String Pattern

String + special character combinations:

- **`?`** - The character before is optional (can be present or not)
- **`+`** - The character before can repeat multiple times
- **`*`** - The begin and end characters need to match it
- **`()`** - Apply with other special characters for multiple character groups
